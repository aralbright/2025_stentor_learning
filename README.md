# Molecular changes during learning in the single cell, _Stentor coeruleus_

This repository accopmanies our 2025 study, "Molecular changes during learning in the single cell, _Stentor coeruleus_." _Stentor_ are giant single-celled protists capable of many complex behaviors, including regeneration and learning. Our study examines the changes in the transcriptome and proteome during learning and forgetting. The file paths in all of these scripts reflect the local environment on Ashley's computer and would have to be changed if you choose to utilize this code.
